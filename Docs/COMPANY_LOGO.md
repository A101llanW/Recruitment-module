# Company Logo Instructions

## Location
Your company logo should be placed in: `HR.Web/Content/images/`

## File Requirements
- **Filename:** `company-logo.png` (replace the existing placeholder)
- **Format:** PNG (recommended) or JPG
- **Size:** 40px height (recommended)
- **Background:** Transparent (recommended for PNG)
- **Resolution:** High quality for crisp display

## Recommended Specifications
- **Height:** 40px
- **Width:** Auto (maintain aspect ratio)
- **Format:** PNG with transparent background
- **File Size:** Under 50KB for optimal loading

## How to Update
1. Replace the current `company-logo.png` file with your actual company logo
2. Refresh the browser to see the changes
3. Test on different screen sizes (mobile and desktop)

## Responsive Behavior
The logo automatically adjusts for different screen sizes:
- **Desktop:** 40px height
- **Mobile:** 30px height
- **Hover effect:** Slight scale animation

## Troubleshooting
If the logo does not appear:
1. Check filename is exactly `company-logo.png`
2. Check path is `Content/images/`
3. Clear browser cache
4. Check file permissions for the web server
